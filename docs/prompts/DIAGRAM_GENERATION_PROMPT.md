# 図解作成用プロンプト集

本ドキュメントは、Figma、Draw.io、PowerPoint、Canva、またはAI画像生成ツール（DALL-E、Midjourney等）で
システム概要図を作成するための詳細な指示書です。

---

## 📌 全体の設計方針

### カラーパレット

| 用途 | 色 | Hex | 使用箇所 |
|-----|-----|-----|---------|
| フロントエンド | オレンジ | `#FF9800` | Next.js関連 |
| バックエンド | グリーン | `#4CAF50` | Python関連 |
| 外部サービス | ピンク | `#E91E63` | API、クラウド |
| ユーザー | ブルー | `#2196F3` | 人物、ブラウザ |
| 強調 | パープル | `#9C27B0` | 重要ポイント |
| 背景 | ライトグレー | `#F5F5F5` | 全体背景 |
| テキスト | ダークグレー | `#424242` | 本文 |

### フォント
- タイトル: Noto Sans JP Bold, 24pt
- 見出し: Noto Sans JP Medium, 18pt
- 本文: Noto Sans JP Regular, 14pt
- コード: Source Code Pro, 12pt

### アイコンスタイル
- 線幅: 2px
- 角丸: 8px
- シャドウ: 0 2px 8px rgba(0,0,0,0.1)
